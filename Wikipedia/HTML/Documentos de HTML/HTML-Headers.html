¿Qué es HTML?

HTML es un lenguaje denominado de "marcación" que sirve para estructurar una página.

HTML se refiere a Hypertext M>arkup Language, como se mencionó, es un lenguaje marcado.

La estructura para HTM L es la siguiente:

  1. Apertura de la etiqueta
  2. Contenido de la etiqueta
  3. Cierre de la etiqueta

La apertura debe seguir este método:

< nombre_de_etiqueta >

Una vez que se definió la etiqueta que será usada, el contenido puede ser texto o puede ser otra etiqueta anidada:

< /nombre_de_etiqueta >

Un ejemplo de código de etiquetas es el siguiente:

<h1> Esto es un título </h1>

Para comentar algo en HTML es así <!--Este código está comentado-->

Etiquetas de HTML

Algunas de las etiquetas más comunes son:

  • <h1> define el título principal de la página que se interpreta como header, se corresponde con lo siguiente otros ejemplos son los siguientes:
  • <H2>
  • <H3>
  • <H4>
  • <H5>
  • <H6>
  • <strong> para poner el texto en negritas
  • <em> para poner el texto en itálicas que proviene de emphasis
  • <p> es un bloque de HTML que define un párrafo
  • <code> es un bloque que define un snippet
  • <span> es una etiqueta genérica
  • <div> la más genérica de todas las etiquetas
  • <header> delimita la sección de una página
