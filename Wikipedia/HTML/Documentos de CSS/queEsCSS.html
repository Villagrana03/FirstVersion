CSS es una hoja de estilo disenado para darle forma visual a una pagina web. CSS hace referencia a Cascade Style Sheet

se denomina hoja de estilo en cascada por la forma que se estructura el codigo de un archivo CSS.

Existen 3 maneras de trabajar CSS.

  • Inline CSS
  • CSS interno
  • CSS externo

Inline CSS EL CSS que se trabaja en forma de Linea, se escribe spbre la etiqueta de HTML a traves del atributo Style.

CSS internoSe trabaja por medio de la etiqueta <Style > Dentro de esta etiqueta se escribe el codigo de CSS.

CSS Externo: El codigo de CSS se trabaja por medio de un archivo CSS externo que se manda a llamar al HTML que necesita los estilos a traves de la etiqueta <Link>

Estructura del codigo CSS

EL codigo CSS se estructura de la siguiente manera:

em 
        /* Forma generica */
        selector{
            atributo: valor;
        }

        /* Selector de una ID */
        #selector{
            atributo: valor;
        }

        /* Selector de una clase */
        .selector{
            atributo: valor;
        }

En CSS los selectores pueden ser una etiqueta de HTML, un ID, estilo o una Clase. Como se muestra a continuacion

        < H1 id="ejemploID" >
        < H1 clase="ejemploClase" >
        < H1 style="atributo: valor;" >
