Types of variables.

There are 3 types of variables in JS

Let Introduced in ES6, let allows you to declare block-scoped variables. Variables declared with let have scope limited to the block they are defined in

let age = 19

Const introduced in ES6, const is used to declare variables that cannot be reassigned after their initial assignment.

const age = 19

Var Before the introduction of ES6, var was the primary keyword used to declare variables. It has function scope or global scope

var age = 19

For Loop

A for loop is a programming construct used for repetitive execution of a block of code.

for( let i = 0; i < array.length; i++)

for(Initialization; Condition; Iteration)

For loop Parameters

the for loop takes 3 parameters, which are for(initialization; condition; iteration)

  1. Initialization: The initialization parameter is executed before the loop begins. It is used to initialize the loop variable or any other necessary variables. This parameter typically assigns an initial value to the loop variable.
  2. Condition: The condition parameter is evaluated before each iteration of the loop. If the condition is true, the loop continues executing. If the condition is false, the loop terminates
  3. Iteration: The iteration parameter is executed at the end of each iteration, just before evaluating the condition again. It typically modifies the loop variable

While Loop

In JavaScript, a while loop is a control flow statement that allows you to repeatedly execute a block of code as long as a specified condition evaluates to true.

What's an array?

Array is a data structure that allows you to store and organize multiple values of the same data type in a single variable

    Examples:

  1. Grouping Data: Arrays are used to organize related data elements together. For example, if you have a list of students' names

  2. Iteration and Loops: Arrays work seamlessly with loops, allowing you to iterate over all elements of an array and perform operations on each element without writing repetitive code.

  3. Implementing Data Structures: Many data structures, such as stacks, queues, and matrices, are often implemented using arrays.
